Conversor de Unidades

Projeto desenvolvido em Python utilizando a biblioteca CustomTkinter para criar uma interface gráfica moderna e intuitiva para conversão de unidades.

Sobre o Projeto

Este projeto foi inspirado em um desafio de desenvolvimento de um conversor de unidades. A proposta original era criar uma aplicação web, porém o projeto está sendo desenvolvido como uma aplicação desktop utilizando Python e CustomTkinter.

O objetivo é permitir que o usuário informe um valor, escolha a unidade de origem e a unidade de destino, realizando a conversão de forma simples e rápida.
